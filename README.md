# Application de Cryptographie

Application graphique complète pour l'apprentissage de la cryptographie.

## Algorithmes implémentés

### Chiffrements classiques
- ✅ César
- ✅ Affine
- ✅ Vigenère
- ✅ Porta
- ✅ Substitution (alphabets désordonnés)
- ✅ Transposition (columaire et Rail Fence)
- ✅ ADFGVX

### Chiffrements modernes
- ✅ RSA
- ✅ AES

### Cryptanalyse
- ✅ Analyse de fréquence
- ✅ Force brute (César)
- ✅ Test du Chi²
